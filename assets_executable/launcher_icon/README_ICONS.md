We can put additional icon sizes into this directory. Missing sizes will be automatically generated.

The following icon sizes are supported:

Windows:
-

| Size    | Filename  |
|---------|-----------|
| 256x256:| `256.png` |
| 128x128:| `128.png` |
| 64x64:  | `64.png`  |
| 48x48:  | `48.png`  |
| 32x32:  | `32.png`  |
| 16x16:  | `16.png`  |

Android:
-

| Size    | Filename  |
|---------|-----------|
| 512x512:| `512.png` |
| 192x192:| `192.png` |
| 144x144:| `144.png` |
| 96x96:  | `96.png`  |
| 72x72:  | `72.png`  |
| 48x48:  | `48.png`  |
